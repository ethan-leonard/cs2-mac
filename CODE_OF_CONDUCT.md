# Code of Conduct

## Our Pledge

We are committed to keeping this project welcoming, respectful, and useful for
everyone. We expect contributors, maintainers, and users to treat one another
with courtesy and good faith.

## Expected Behavior

- Be respectful in issues, pull requests, reviews, and chat.
- Focus feedback on the code and the problem, not the person.
- Share credit and be clear when you are unsure about something.
- Assume good intent unless there is strong evidence otherwise.

## Unacceptable Behavior

- Harassment, discrimination, or threats
- Personal attacks or repeated rude behavior
- Doxxing, stalking, or intimidation
- Deliberate misinformation or bad-faith disruption

## Scope

This policy applies in project repositories, issue trackers, pull requests,
discussion forums, and any project spaces where this community interacts.

## Enforcement

Maintainers may remove content, lock threads, or limit participation when
behavior violates this code of conduct.

If you need to report a concern, use the repository host's private maintainer
contact path or security reporting channel where available.
